Options

Add new option in HTML: At end of last upper element add comma, and than add new option in format option : value.

slideHeight 560 or "560px" Height of slider. Height can be integer (1000) or string in pixels ("1000px"), percent ("100%") and text ("auto").

transitionType "slide" Select transition type. Can be "fade" or "slide".

transitionSpeed 400 Time in milliseconds. Transition animation duration.

enable_rightClick false This option disables mouse right click on images. To enable, write true.
enable_imageDrag false This option disables image dragging with mouse. To enable, write true.

retina_HD_enable false To activate type true. More in Retina & HD section.

slide_title_anim "drop" Come-in animation for description title. Available: none (without animation), blind, bounce, clip, drop, explode, fade, fold, highlight, puff, pulsate, scale, shake, size, slide, transfer. See description section.

Images

Some word about autofit option:
By default autofit option is enabled (see options). This option always check slider dimensions and fulfill him with image by height or width. By default in both situations image is centered by height (if image is higher than slider) or width (if image is wider than slider). If you don't see part of image which you want, you can "move" image by height (up or down) or width (left or right). More in section below. If you don't want autofit, turn him off and than things become more simple. Images in that situation are on 100% of width. If you want you can do it by height too. Just add in options:

Retina & HD

What is retina? Retina is just name for device pixel ratio. More at wikipedia. Like option in slider, retina is for better image quality on displays with bigger pixel ratio that 1.0 (1.0 is normal computer display). If you have image for ratio 1.0, on displays with bigger ratio image can look awfully. So here comes this option which replace normal images with "retina" images. By default retina/HD is disabled. Type true in intialize to turn on. "Retina" images are images which are bigger than normal. The best way is to use double sized images (ie if you have normal image with dimensions 500x250 pixels, use for retina image with dimensions 1000x500 pixels). And what about HD (High Definition)? Retina and HD works on the same priciple. Both uses same images from same folder. Only difference is that HD depends on slider dimensions. If slider dimensions are over 1500x845 pixels, slider will use "retina", ie HD images. I will continue talk about retina and HD images, but i will use only retina word for simplicity. You can add this "retina" images in same folder as normal ones or you can put them in some other folder. Example how looks options in initialize:
If you have retina images in same folder as normal ones, by standards, they must have some other name. For that you have retina_text option. So, images must have that text in name too. Example: if you have normal image named image1.jpg, retina image will be image1_retina.jpg. Option retina_text is optional if you have images in another folder. For that situation you can leave just "". Same is with retina_folder. Its optional if images have another name, ie if they have retina_text. You can leave too just "". But both can't be with "".

Customization

Change rotated » "more info" text

So, if you wanna change rotated » which we found in description title you have to add in initialize section two options (options are in options section):
Option more_info_text is text which will be shown instead of ». Option rotation_by_deg when is activated (true) rotates text for 90 degrees when description is up and for -90 degress (ie 270 degrees) when is down. So, when you have some text instead of » there is no need for rotation. Just type false to deactivate this option. But, text will be large. In CSS file find .more_info class style and change it: 

        .more_info {

			/* some style before. Change it only if you know */

			font-size:26px; /* change font height (size) */

			font-weight:bold; /* change to normal if you don't want fat font :) */

		}
